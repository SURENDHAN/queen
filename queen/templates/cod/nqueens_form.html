Introduction

The N-Queens problem is a classic combinatorial problem that involves placing N queens on an N x N chessboard so that no two queens threaten each other. This means that no two queens can be in the same row, column, or diagonal.

This problem is a great exercise for learning about algorithms and problem-solving strategies. It has practical applications in various fields such as artificial intelligence and optimization.

How It Works

The N-Queens problem can be solved using different algorithms. The most common method is the backtracking algorithm. Here's a brief overview:

  • Backtracking: This approach involves placing queens one by one in different columns and rows. If a conflict is detected (i.e., two queens threaten each other), the algorithm backtracks to the previous step and tries a different configuration.
  • Constraint Propagation: This method involves reducing the search space by propagating constraints throughout the board. It simplifies the problem and speeds up the solution process.

Code Example

Below is a Python example of a backtracking solution for the N-Queens problem:


def solve_n_queens(n):
    def is_valid(board, row, col):
        for i in range(row):
            if board[i] == col or \
               board[i] - i == col - row or \
               board[i] + i == col + row:
                return False
        return True

    def solve(row, board):
        if row == n:
            results.append(board[:])
            return
        for col in range(n):
            if is_valid(board, row, col):
                board[row] = col
                solve(row + 1, board)
                board[row] = -1

    results = []
    solve(0, [-1] * n)
    return results

# Example usage
n = 8
solutions = solve_n_queens(n)
print(f"Number of solutions for {n}-Queens: {len(solutions)}")
for sol in solutions:
    print(sol)

This code defines a function solve_n_queens that uses the backtracking approach to find all solutions for placing N queens on an N x N chessboard.
